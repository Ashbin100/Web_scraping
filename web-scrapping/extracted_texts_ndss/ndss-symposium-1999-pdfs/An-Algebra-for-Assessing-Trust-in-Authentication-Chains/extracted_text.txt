19Concluding remarks
(cid:15)
The presented trust model is more complete than
previously proposed models because it can
express degrees of uncertainty.
(cid:15)
Subjective Logic can be used directly for reason-
ing about trust in practical security applications.
(cid:15)
A key certificate must contain:
1) recommendation about key authenticity,
2) recommendation about key owner.
(cid:15)
Recommendation of trust must be based on
first-hand evidence only.
